## 🧮 알고리즘 (미국 버전과 동일 코어)

### Robust Z-score
```
z = (x - median) / (1.4826 × MAD)
±5σ winsorization 적용
```

### 5 Valuation Multiples
- P/E (Net Income/Market Cap 기준 자체 계산)
- P/S (yfinance 제공)
- P/B (Equity/Market Cap 기준 자체 계산)
- EV/EBITDA (yfinance 제공)
- EV/Sales (yfinance 제공)

### Piotroski F-Score (Quality)
9개 이진 지표 합산 (0~9):
- Profitability (4): ROA>0, CFO>0, ΔROA>0, CFO>NI
- Leverage/Liq (3): ΔDebt↓, ΔCurrentRatio↑, No dilution
- Operating (2): ΔGrossMargin↑, ΔAssetTurnover↑

### 12-1M Momentum
Jegadeesh & Titman (1993). 단기 reversal 제거.

### Multi-Factor Composite
```
final_score = w_v · rank(value) + w_q · rank(quality) + w_m · rank(momentum)
```
가중치는 `ai_semi_phase`에 따라 동적 조정 (Late Bull 시 Quality 50%).

### Bayesian Shrinkage
작은 섹터(N<10)는 전체 평균으로 shrink, 과적합 방지.

### Sector-Neutral Long-Short
각 섹터에서 quintile 분류, Top 20% Long / Bottom 20% Short.

## ⚠️ 주의사항 (한국 시장 특수성)

1. **yfinance 한국 종목 데이터 품질**: 미국 종목 대비 누락 많음. trailingPE / priceToBook는 직접 계산.
2. **Korea Discount**: 같은 비즈니스도 미국 대비 30~50% 낮은 멀티플이 정상. 절대값으로 비교 X
3. **외국인 매도 압력**: USD/KRW 1300+ 구간에서 외국인 매도 빈번 → momentum 시그널 약화
4. **분기 실적 공시 시즌**: 1월/4월/7월/10월 → 펀더멘털 급변 가능
5. **재벌 그룹 리스크**: 지주사 디스카운트, 일가 관련 이슈로 단기 변동 큼
